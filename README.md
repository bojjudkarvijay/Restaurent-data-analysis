# 🍽️ Tips Data Analysis (EDA Project)

## 📌 Overview

This project performs Exploratory Data Analysis (EDA) on the popular **Tips dataset** using Python libraries like Pandas, Seaborn, and Matplotlib.

## 📊 Features

* Data cleaning and inspection
* Statistical summary
* Visualization of:

  * Total bill vs Tip
  * Tips by Gender
  * Tips by Day
  * Tips by Time
  * Group size vs Tip
* Correlation heatmap

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## ▶️ How to Run

```bash
pip install -r requirements.txt
python analysis.py
```

## 📈 Results

* Higher bills generally lead to higher tips
* Dinner tips are higher than lunch
* Weekend tips are higher
* Group size affects tip amount

## 📁 Dataset

Built-in dataset from Seaborn
